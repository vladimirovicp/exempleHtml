Разделение - это микроблокнот JavaScript с набором небольших встроенных плагинов, предназначенных для разделения (разделения) элемента различными способами, такими как слова, символы, дочерние узлы и многое другое!

Библиотека разделения не обрабатывает анимацию, но предоставляет вам элементы и инструменты, необходимые для создания анимаций и переходов с помощью библиотек анимации JavaScript или только CSS!. Большинство плагинов используют набор <span> символов, заполненных CSS-переменными и атрибутами данных, которые позволяют создавать всевозможные анимации, переходы и взаимодействия.
